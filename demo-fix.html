LivePhoto Web Component

Apple-style Live Photos for any website

Example 1: Muted (Default)

Hover the LIVE badge on desktop, or long-press on mobile.

Example 2: With Sound

Set muted="false" to enable audio.

Example 3: Full Width

Omit width or set to 100% for responsive layout.

How to Use

<script src="dist/live-photo.umd.js"></script>
<link rel="stylesheet" href="dist/styles.css">

<live-photo
  photo="image.jpg"
  video="video.mp4"
  width="600"
  muted>
</live-photo>
